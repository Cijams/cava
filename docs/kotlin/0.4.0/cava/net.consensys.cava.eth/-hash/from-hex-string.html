static fun fromHexString(str: String): Hash (source)

Parse a hexadecimal string into a Hash.

Parameters

str - The hexadecimal string to parse, which may or may not start with "0x". That representation may contain less than 32 bytes, in which case the result is left padded with zeros.

Exceptions

IllegalArgumentException - if str does not correspond to a valid hexadecimal representation or contains more than 32 bytes.

Return
The value corresponding to str.
