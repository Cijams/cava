open static fun wrap(value: ByteArray, offset: Int): MutableBytes32 (source)

Wrap a the provided array as a MutableBytes32.

Note that value is not copied, only wrapped, and thus any future update to value within the wrapped parts will be reflected in the returned value.

Parameters

value - The bytes to wrap.

offset - The index (inclusive) in value of the first byte exposed by the returned value. In other words, you will have wrap(value, i).get(0) == value[i].

Exceptions

IndexOutOfBoundsException - if offset &lt; 0 || (value.length &gt; 0 && offset >= value.length).

IllegalArgumentException - if length &lt; 0 || offset + 32 &gt; value.length.

Return
A MutableBytes32 that exposes the bytes of value from offset (inclusive) to offset + 32 (exclusive).

Wrap a slice/sub-part of the provided value as a MutableBytes32.

Note that the value is not copied, and thus any future update to value within the wrapped parts will be reflected in the returned value.

Parameters

value - The bytes to wrap.

offset - The index (inclusive) in value of the first byte exposed by the returned value. In other words, you will have wrap(value, i).get(0) == value.get(i).

Exceptions

IndexOutOfBoundsException - if offset &lt; 0 || (value.size() &gt; 0 && offset >= value.size()).

IllegalArgumentException - if length &lt; 0 || offset + 32 &gt; value.size().

Return
A Bytes32 that exposes the bytes of value from offset (inclusive) to offset + 32 (exclusive).
