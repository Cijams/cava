readULongIntList

open fun readULongIntList(bitLength: Int): MutableList<Long> (source)

Read a list of unsigned long int values from the SSZ source.

Parameters

bitLength - The bit length of the integers to read (a multiple of 8).

Exceptions

InvalidSSZTypeException - If the next SSZ value is not a list, there are insufficient encoded bytes for the desired bit length or any value in the list, or any decoded value was too large to fit into a long.

EndOfSSZException - If there are no more SSZ values to read.

Return
A list of longs.
