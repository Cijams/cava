readBytes

open fun readBytes(): Bytes (source)

Read bytes from the SSZ source. Note: prefer to use #readBytes(int) instead, especially when reading untrusted data.

Exceptions

InvalidSSZTypeException - If the next SSZ value is not a byte array, or is too large (greater than 2^32 bytes).

EndOfSSZException - If there are no more SSZ values to read.

Return
The bytes for the next value.
