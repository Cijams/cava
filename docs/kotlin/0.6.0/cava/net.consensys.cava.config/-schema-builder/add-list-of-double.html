addListOfDouble

fun addListOfDouble(key: String, @Nullable defaultValue: MutableList<Double>?, @Nullable description: String?, @Nullable validator: PropertyValidator<in MutableList<Double>>?): SchemaBuilder (source)

Add a list-of-doubles property to the schema.

Even if no validator is provided, the schema will validate that the configuration property, if present, contains a list of doubles without any null values.

If a defaultValue is provided, then the provided validator, if any, will only be invoked if the value is present (i.e. it will not be provided a null value to validate).

Parameters

key - The configuration property key.

defaultValue - A default value for the property or null if no default is provided.

description - The description to associate with this property, or null if no documentation is provided.

validator - A validator for the property, or null if no validator is provided.

Exceptions

IllegalArgumentException - If the key cannot be parsed.

Return
This builder.
