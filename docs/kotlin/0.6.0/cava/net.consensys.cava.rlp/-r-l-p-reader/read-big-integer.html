open fun readBigInteger(lenient: Boolean): BigInteger (source)

Read a big integer value from the RLP source.

Parameters

lenient - If false, an exception will be thrown if the integer is not minimally encoded.

Exceptions

InvalidRLPEncodingException - If there is an error decoding the RLP source, or the integer is not minimally encoded and `lenient` is false.

InvalidRLPTypeException - If the next RLP value cannot be represented as a big integer.

EndOfRLPException - If there are no more RLP values to read.

Return
A big integer.
