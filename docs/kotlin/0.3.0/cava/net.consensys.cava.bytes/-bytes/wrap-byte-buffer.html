open static fun wrapByteBuffer(byteBuffer: ByteBuffer, offset: Int, size: Int): Bytes (source)

Wrap a slice of a Java NIO ByteBuf as a Bytes value.

Note that any change to the content of the buffer may be reflected in the returned value.

Parameters

byteBuffer - The ByteBuffer to wrap.

offset - The offset in byteBuffer from which to expose the bytes in the returned value. That is, wrapByteBuffer(byteBuffer, i, 1).get(0) == byteBuffer.getByte(i).

size - The size of the returned value.

Exceptions

IndexOutOfBoundsException - if offset &lt; 0 || (byteBuffer.limit() > 0 && offset >= byteBuf.limit()).

IllegalArgumentException - if length &lt; 0 || offset + length > byteBuffer.limit().

Return
A Bytes value.
